# LoveShip

This is a simple game written for the [Love2D framework][love] and the [Dream.In.Code][dic] _52 Weeks of Code_ challenges.

## What there is so far:
 * A ship that flies around a star field. Weeeeee!

## Next goal:
 * Limited fuel
 * Fuel refills
 * Points

## The future:
 * Radar
 * Weapons
 * Enemies / Obstacles 
 * Upgrades
